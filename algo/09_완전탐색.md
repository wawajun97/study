# 완전탐색

완전탐색은 가능한 모든 경우의 수를 확인하는 기법이다.

- 일반적으로 경우의 수가 상대적으로 적을 때 사용한다.
- 적용하기 전에는 시간 복잡도를 먼저 계산해 볼 필요가 있다.

## 순열과 조합

### 순열

순열은 서로 다른 `n`개 중에서 `r`개를 뽑아 나열하는 경우의 수이다.

- 표기: `nPr`
- 순서가 중요하다.

### 예시: 서로 다른 3개의 수로 3자리 만들기

- 첫 번째 자리는 3가지 경우가 있다.
- 두 번째 자리는 이미 뽑은 수를 제외하므로 2가지 경우가 있다.
- 세 번째 자리는 남은 1가지만 가능하다.
- 따라서 `3 * 2 * 1 = 6`가지 경우의 수가 나온다.
- 재귀를 사용해 구현할 수 있다.

```java
public static void perm(int cnt) {
		if(cnt == 3) {
			//순열 생성 완료 -> 비즈니스 로직 처리
			return;
		}

		for(int i = 1;i<=3;i++) {
			if(isSelected[i]) continue;

			numbers[cnt] = i;
			isSelected[i] = true;
			perm(cnt+1);
			isSelected[i] = false;
		}
	}
```

### 중복순열

중복순열은 중복을 허용하는 순열이다.

- 같은 숫자를 여러 번 사용할 수 있다.
- 따라서 `isSelected` 배열이 필요 없다.

```java
public static void dice(int depth) {
		if(depth == N) {
			// System.out.println(Arrays.toString(numbers));

			for(int i = 0;i<N;i++) {
				System.out.print(numbers[i] + " ");
			}
			System.out.println();
			totalCnt++;
			return;
		}

		for (int i = 1; i <= 6; i++) { //숫자 사용 체크 x
			numbers[depth] = i;
			dice(depth+1);
		}
	}
```

### 조합

조합은 서로 다른 `n`개 중에서 순서와 상관없이 `r`개를 뽑는 경우의 수이다.

- 표기: `nCr`
- 순서가 중요하지 않다.

```java
public static void dice3(int depth, int cur) {
		if(depth == N) {
//			System.out.println(Arrays.toString(numbers));

			for(int i = 0;i<N;i++) {
				System.out.print(numbers[i] + " ");
			}
			System.out.println();
			totalCnt++;
			return;
		}

		for (int i = cur; i <= 6; i++) { //시작이 중요
			numbers[depth] = i;
			dice3(depth+1,i+1);
		}
	}
```

### 중복조합

중복조합은 중복을 허용하는 조합이다.

- 예를 들어 `1 1 2`는 가능하다.
- 하지만 `1 2 1`은 같은 조합으로 본다.

```java
public static void dice3(int depth, int cur) {
		if(depth == N) {
//			System.out.println(Arrays.toString(numbers));

			for(int i = 0;i<N;i++) {
				System.out.print(numbers[i] + " ");
			}
			System.out.println();
			totalCnt++;
			return;
		}

		for (int i = cur; i <= 6; i++) {
			numbers[depth] = i;
			dice3(depth+1,i); //cur 파라미터를 +1 해주지 않으면 중복조합
		}
	}
```

## 부분집합

- 부분집합은 집합에 포함된 원소들 중 일부를 선택해 만드는 집합이다.
- 순서가 중요하지 않다는 점에서 조합과 비슷하다.
- 예를 들어 배낭 문제처럼 최적의 부분집합을 찾는 문제에서 자주 사용된다.

## 생성 방법

### 1. 반복문

```java
public static void makeSubSet() {
		int[] selected = new int[3];
		int[] arr = {1,2,3};

		for(int i = 1;i>=0;i--) { //1은 선택 0은 비선택
			selected[0] = i;
			for(int j = 1; j>=0;j--) {
				selected[1] = j;
				for(int k = 1; k>=0;k--) {
					selected[2] = k;
					for(int m = 0;m<3;m++) {
						if(selected[m] == 1) { //선택이 된 idx만 출력
							System.out.print(arr[m] + " ");
						}
					}
					System.out.println();
				}
			}
		}
	}
```

### 2. 재귀

```java
public static void makeSubSet(int idx) {
		if(idx == N) {
			for(int i = 0;i<N;i++) {
				if(isSelected[i]) {
					System.out.print(input[i] + " ");
				}
			}
			System.out.println();
			return;
		}

		//idx의 원소를 선택하는 경우
		isSelected[idx] = true;
		makeSubSet(idx+1);

		//idx의 원소를 선택하지 않는 경우
		isSelected[idx] = false;
		makeSubSet(idx+1);
	}
```
