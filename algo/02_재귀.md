재귀 : 함수 내부에서 직접 혹은 간접적으로 자기 자신을 호출하는 함수

기본 부분 : 더 이상 재귀가 파생되지 않는 부분 -> 가장 작은 문제가 되는 출발점
유도 부분 : 재귀가 파생되는 부분

1. 함수의 정의를 명확히 하자
2. 기능 구현 설계를 할 때 자신의 작업과 나머지 작업으로 나누어 설계하자
    -> 하나의 함수에서 다하려고 하지 말자
    -> 부분문제 식별
3. 함수의 수행을 결정짓는 결정적 요인이 되는 매개체가 되는 값 식별
   -> 매개변수 식별
4. 재귀의 기본부분을 파악
   -> 재귀의 탈출/종료 조건

```
public static void main(String[] args) {
		test(4);
	}

	private static void test(int size) {
		if(size == 0) return; //base case
		
		test(size-1);
		
		for(int i = 0; i <size;i++) {
			System.out.print("* ");
		}
		System.out.println();
	}
```
