# 최단 경로

## 다익스트라 알고리즘

다익스트라 알고리즘은 가중치가 있는 그래프에서 최단 경로를 찾는 알고리즘이다.

- 음수 가중치는 허용하지 않는다.
- 음수 가중치가 있다면 보통 벨만-포드 알고리즘을 사용한다.

### 기본 아이디어

1. 시작점에서 가장 가까운 정점을 선택한다.
2. 그 정점을 거쳐 다른 정점으로 가는 비용을 갱신한다.
3. 이를 반복하면서 최단 거리를 확정해 나간다.

## 인접 리스트 기반 구현

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.Arrays;
import java.util.StringTokenizer;

public class Dijkstra_AdjList {
	public static class Node {
		int vertex;
		int weight;
		Node next;

		public Node(int vertex, int weight, Node next) {
			this.vertex = vertex;
			this.weight = weight;
			this.next = next;
		}
	}
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine(), " ");

		int V = Integer.parseInt(st.nextToken());
		int E = Integer.parseInt(st.nextToken());

		st = new StringTokenizer(br.readLine(), " ");
		int start = Integer.parseInt(st.nextToken());
		int end = Integer.parseInt(st.nextToken());

		Node[] adjList = new Node[V];
		boolean[] visited = new boolean[V];
		int[] minDist = new int[V];

		for(int j =0;j<E;j++) {
			st = new StringTokenizer(br.readLine(), " ");
			int from = Integer.parseInt(st.nextToken());
			int to = Integer.parseInt(st.nextToken());
			int weight = Integer.parseInt(st.nextToken());
			adjList[from] = new Node(to,weight,adjList[from]);
		}

		final int INF = Integer.MAX_VALUE;

		Arrays.fill(minDist, INF);

		minDist[start] = 0;

		for(int i = 0;i<V;i++) {
			int cur = -1;
			int min = INF;

			for(int j = 0;j<V;j++) {
				if(!visited[j] && min > minDist[j]) {
					cur = j;
					min = minDist[j];
				}
			}

			if(cur == -1) break;

			visited[cur] = true;

			for (Node temp = adjList[cur]; temp != null ; temp = temp.next) {
				if(!visited[temp.vertex] && minDist[temp.vertex] > min + temp.weight) {
					minDist[temp.vertex] = min + temp.weight;
				}
			}
		}

		System.out.println(Arrays.toString(minDist));
		System.out.println(minDist[end] != INF ? minDist[end] : -1);
	}

}
```

## PriorityQueue 사용

우선순위 큐를 사용하면 더 효율적으로 구현할 수 있다.

```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.Arrays;
import java.util.PriorityQueue;
import java.util.StringTokenizer;

public class Dijkstra_PQ {
	public static class Node {
		int to;
		int weight;
		Node next;

		public Node(int to, int weight, Node next) {
			this.to = to;
			this.weight = weight;
			this.next = next;
		}
	}

	public static class Vertex implements Comparable<Vertex> { //PQ에 넣을 정보
		int no, dist;

		public Vertex(int no, int dist) {
			super();
			this.no = no;
			this.dist = dist;
		}

		@Override
		public int compareTo(Vertex o) {
			return Integer.compare(this.dist, o.dist);
		}
	}
	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = new StringTokenizer(br.readLine(), " ");

		int V = Integer.parseInt(st.nextToken());
		int E = Integer.parseInt(st.nextToken());

		st = new StringTokenizer(br.readLine(), " ");
		int start = Integer.parseInt(st.nextToken());
		int end = Integer.parseInt(st.nextToken());

		Node[] adjList = new Node[V];
		int[] minDist = new int[V];

		for(int j =0;j<E;j++) {
			st = new StringTokenizer(br.readLine(), " ");
			int from = Integer.parseInt(st.nextToken());
			int to = Integer.parseInt(st.nextToken());
			int weight = Integer.parseInt(st.nextToken());
			adjList[from] = new Node(to,weight,adjList[from]);
		}

		PriorityQueue<Vertex> pq = new PriorityQueue<>();
		final int INF = Integer.MAX_VALUE;
		Arrays.fill(minDist, INF);

		minDist[start] = 0;
		pq.offer(new Vertex(start,minDist[start]));

		while(!pq.isEmpty()) {
			Vertex cur = pq.poll();

			if(minDist[cur.no] < cur.dist) continue;

//			if(cur.no == end) break;

			for (Node temp = adjList[cur.no]; temp != null ; temp = temp.next) {
				if(minDist[temp.to] > cur.dist + temp.weight) { // cur.dist = minDist[cur.no]
					minDist[temp.to] = cur.dist + temp.weight;
					pq.offer(new Vertex(temp.to, minDist[temp.to]));
				}
			}
		}

		System.out.println(Arrays.toString(minDist));
//		System.out.println(minDist[end] != INF ? minDist[end] : -1);
	}

}
```

## 0-1 BFS

0-1 BFS는 간선 가중치가 0 또는 1인 경우에 사용할 수 있는 최단 경로 알고리즘이다.

- `Deque`를 사용해 구현한다.
- 가중치가 0이면 앞쪽(`front`)에 넣는다.
- 가중치가 1이면 뒤쪽(`rear`)에 넣는다.
- 이렇게 하면 항상 더 작은 비용의 경로를 먼저 처리할 수 있다.
