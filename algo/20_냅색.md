# 배낭 문제

배낭 문제는 각 물건의 무게와 가치가 주어졌을 때, 제한된 무게 안에서 담을 수 있는 최대 가치를 구하는 문제이다.

## 1. 0-1 Knapsack

- 물건을 쪼갤 수 없고 통째로 담아야 하는 문제이다.

가능한 그리디 접근은 다음과 같지만, 일반적으로는 최적해를 보장하지 않는다.

- 가치가 큰 물건부터 담기
- 가벼운 물건부터 담기
- 무게 대비 가치가 높은 순서로 담기

## 2. Fractional Knapsack

- 물건을 부분적으로 잘라 담을 수 있는 문제이다.
- 이 경우에는 무게 대비 가치가 높은 순서로 담는 그리디가 성립한다.

## DP 관점에서의 0-1 Knapsack

- `k[i][w]`: `i`번째 물건까지 고려했을 때, 무게 `w`를 만족하는 최대 가치
- `i == 0` 또는 `w == 0`이면 값은 0이다.
- 현재 물건을 담는 경우와 담지 않는 경우 중 더 큰 값을 선택한다.

```java
//2차원 배열 사용 시
int[][] dp = new int[n+1][k+1];
for(int i =1;i<=n;i++) {
  for (int j = 1; j <= k; j++) {
    if(w[i] > j) {
      dp[i][j] = dp[i-1][j];
    } else {
      dp[i][j] = Math.max(dp[i-1][j-w[i]] + v[i], dp[i-1][j]);
    }
  }
}

//1차원 배열 사용 시
int[] dp = new int[k+1];
for(int i =1;i<=n;i++) {
  for (int j = k; j >= w[i]; j--) { //앞에서부터 채우면 물건이 중복으로 들어가서 뒤에서부터 채워넣기
    dp[j] = Math.max(dp[j-w[i]] + v[i], dp[j]);
  }
}
```

## 최장 증가 부분 수열(LIS)

예: `[3,2,6,4,5,1]`에서 원래 순서를 유지하면서 값이 점점 커지는 가장 긴 부분 수열의 길이를 구한다.

### 1. DP - `O(N^2)`

- `LIS(i)`: `a[i]`를 마지막 원소로 하는 LIS 길이
- `a[i]`보다 작은 `a[0] ~ a[i-1]` 중 가장 긴 수열에 1을 더한 값이 된다.

```java
int max = 1;
for(int i = 0;i<n;i++) {
  dp[i] = 1;

  for(int j = 0;j<i;j++) {
    if(arr[j] < arr[i] && dp[i] < dp[j] + 1) {
      dp[i] = dp[j] + 1;
      max = Math.max(max, dp[i]);
    }
  }
}
```

### 2. DP + 이진 탐색 - `O(NlogN)`

- `C[k]`: 길이 `k`인 증가 수열에서 마지막 위치에 올 수 있는 값들 중 가장 작은 값
- 각 원소를 볼 때마다 이진 탐색으로 들어갈 위치를 찾고 갱신한다.

```java
memo = new int[n+1]; //memo[i]는 i번째로 올 수 있는 수 중 가장 작은 수
int len=0;
int idx=0;
for(int i=0; i<n; i++) {
  if(arr[i] > memo[len]) { //맨 뒤에 오는 수 이므로 len++
    len +=1;
    memo[len] = arr[i];
  }else { //여기 걸리는 arr[i] 이후로 생성되는 배열이 최장일 수 있어서 memo[idx]를 바꿔줘야함 -> 중간에 수를 바꿔서 최장 길이는 그대로
    idx = binarySearch(0,len, arr[i]);
    memo[idx] = arr[i];
  }
}
System.out.println(len);

//arr[i]를 넣을 idx 찾기
static int binarySearch(int left, int right, int key) {
  int mid =0;
  while(left<right) {
    mid = (left+right)/2;
    if(memo[mid] < key) {
      left = mid+1;
    }else {
      right = mid;
    }
  }
  return right;
}
```
