배낭 채우기 : 각각의 물건은 무게와 가치가 있고 배낭에 특정 무게만큼만 담을 수 있을 때 배낭에 담을 수 있는 물건들의 최대 가치  

1. 0-1 Knapsack  
- 배낭에 물건을 통째로 담아야 하는 문제

방법1 : 값이 비싼 물건부터 채운다.  
방법2 : 무게가 가벼운 물건부터 채운다.  
방법3 : 무게 당 값이 높은 순서로 물건을 채운다.  

2. Fractional Knapsack
- 물건을 잘라서 부분적으로 담을 수 있는 문제  

방법1 : 무게 당 값이 높은 순서로 물건을 채운다.  

=> k[i][w] : i물건까지 고려했을 때 w가방무게를 만족하는 최적 가치  
=> i == 0 || w == 0 이면 0으로 채우기  
=> k[i][w] = Math.max(vi + k[i-1][w-wi], k[i-1][w]) -> 지금 물건을 채우는 경우와 안 채우는 경우를 비교하여 더 큰 경우가 최적
=> 정답 : 마지막 물건의 w 무게. 즉, k[i,w]가 정답  

```java
//2차원 배열 사용 시
int[][] dp = new int[n+1][k+1];
for(int i =1;i<=n;i++) {
  for (int j = 1; j <= k; j++) {
    if(w[i] > j) {
      dp[i][j] = dp[i-1][j];
    } else {
      dp[i][j] = Math.max(dp[i-1][j-w[i]] + v[i], dp[i-1][j]);
    }
  }
}

//1차원 배열 사용 시
int[] dp = new int[k+1];
for(int i =1;i<=n;i++) {
  for (int j = k; j >= w[i]; j--) { //앞에서부터 채우면 물건이 중복으로 들어가서 뒤에서부터 채워넣기
    dp[j] = Math.max(dp[j-w[i]] + v[i], dp[j]);
  }
}
```  

최장 증가 부분 수열  
ex) [3,2,6,4,5,1] 배열 순서를 유지하면서 크기가 점진적으로 커지는 가장 긴 부분 수열의 길이는?  

1. DP - O(N^2)
- LIS(i) : ai를 포함한 LIS 길이
- ai보다 작은 a1 ~ ai-1을 찾고 그 중에서 가장 긴 수열을 가진 LIS() + 1이 LIS(i)의 값
- max = Math.max(max, LIS(i))로 max값 갱신

```java
int max = 1;
for(int i = 0;i<n;i++) {
  dp[i] = 1;
  
  for(int j = 0;j<i;j++) {
    if(arr[j] < arr[i] && dp[i] < dp[j] + 1) {
      dp[i] = dp[j] + 1;
      max = Math.max(max, dp[i]);
    }
  }
}
```

2. DP 이진 검색 - O(NlogN)
- C[k] : 길이 k의 증가 수열에 대하여 길이 k자리에 올 수 있는 가능한 값 중 가장 작은 값을 C[k]에 저장
- 각 원소마다 C[]를 갱신하기 위해 이진 검색 수행

```java
memo = new int[n+1]; //memo[i]는 i번째로 올 수 있는 수 중 가장 작은 수
int len=0;
int idx=0;
for(int i=0; i<n; i++) {
  if(arr[i] > memo[len]) { //맨 뒤에 오는 수 이므로 len++
    len +=1;
    memo[len] = arr[i];
  }else { //여기 걸리는 arr[i] 이후로 생성되는 배열이 최장일 수 있어서 memo[idx]를 바꿔줘야함 -> 중간에 수를 바꿔서 최장 길이는 그대로
    idx = binarySearch(0,len, arr[i]);
    memo[idx] = arr[i];
  }
}
System.out.println(len);

//arr[i]를 넣을 idx 찾기
static int binarySearch(int left, int right, int key) {
  int mid =0;
  while(left<right) {
    mid = (left+right)/2;
    if(memo[mid] < key) {
      left = mid+1;
    }else {
      right = mid;
    }
  }
  return right;
}
```
