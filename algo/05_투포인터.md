투포인터 : 배열이나 리스트에서 두 개의 포인터를 사용해 문제를 해결하는 알고리즘 기법

탐색 방법 예시
1. s,e 두개의 포인터를 양 끝에 위치 시킨다
2. s,e가 가리키는 숫자들이 조건을 만족하는지 확인
   2-1. 만족하지 않으면 s,e 둘 중 하나 위치 이동(조건을 보고 수가 커져야하는지 작아져야 하는지 판단해서)
   2-2. 만족하면 둘 다 이동(숫자가 중복 없을 시)
3. 2번을 반복하다가 s >= e 이면 종료
-> 시간복잡도 = O(N)

```java
public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		
		StringTokenizer st = new StringTokenizer(br.readLine(), " ");
		int N = Integer.parseInt(st.nextToken());
		int target = Integer.parseInt(st.nextToken());
		
		int[] arr = new int[N];
		st = new StringTokenizer(br.readLine(), " ");
		
		for (int i = 0; i < N; i++) {
			arr[i] = Integer.parseInt(st.nextToken());
		}

		Arrays.sort(arr);
		
		int s = 0;
		int e = N - 1;
		int ans = 0;
		
		while(s < e) {
			int sum = arr[s] + arr[e];
			
			if(sum == target) {
				ans++;
				s++;
				e--;
			} else if(sum < target) {
				s++;
			} else {
				e--;
			}
		}
		System.out.println(ans);
	}
```
