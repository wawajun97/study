# 비트 연산자

1. `&`: AND 연산. 둘 다 1일 때만 1
2. `|`: OR 연산. 둘 중 하나라도 1이면 1
3. `^`: XOR 연산. 두 값이 다르면 1, 같으면 0
4. `~`: NOT 연산. 모든 비트를 반전
5. `<<`: 비트를 왼쪽으로 이동. 뒤에 0이 붙으며, 일반적으로 값이 2배씩 커진다.
6. `>>`: 비트를 오른쪽으로 이동. 마지막 비트가 잘려 나간다.

## 비트마스킹

비트 연산은 비트마스킹에 자주 사용된다.

- 하나의 비트로 선택/비선택 같은 2가지 상태를 저장할 수 있다.
- `boolean` 배열로도 가능하지만, 비트마스킹이 메모리 면에서 더 효율적일 수 있다.
- 특정 비트 값을 확인하려면 `1 << n`을 만든 뒤 AND 연산을 하면 된다.
- 특정 비트를 1로 만들려면 `1 << n`을 만든 뒤 OR 연산을 하면 된다.

## 바이너리 카운팅으로 부분집합 생성

- 부분집합을 생성하는 가장 자연스러운 방법 중 하나이다.
- 원소 수만큼의 비트열을 사용한다.
- `n`번째 비트가 1이면 `n`번째 원소가 포함된 것으로 해석한다.

```java
int[] arr = {3,6,7,1,5,4};
		int n = arr.length;

		for(int i = 0;i<(1 << n);i++) {
			for(int j = 0;j<n;j++) {
				if((i & (1<<j)) != 0) {
					System.out.print(arr[j] + " ");
				}
			}
			System.out.println();
		}
```

## BitSet 클래스

- 내부적으로 `long`형 배열을 사용한다.
- 64개 이상의 많은 플래그를 관리할 때 유용하다.

1. 특정 비트 `i`를 토글하려면
   - `flag ^= (1 << i)`
2. 특정 비트 `i`를 0으로 만들려면
   - `flag &= ~(1 << i)`
