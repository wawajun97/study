# 큐

- 큐는 삽입과 삭제가 일어나는 위치가 제한된 선형 자료구조이다.
- 먼저 들어온 데이터가 먼저 나가는 FIFO(First In First Out) 구조를 가진다.
- 자바에서는 삽입과 삭제가 명확한 구조이기 때문에 `ArrayDeque`를 많이 사용한다.

## 기본 연산

- `enQueue`: 삽입
- `deQueue`: 삭제

## front와 rear

- `front`: 맨 앞 원소의 인덱스
- `rear`: 맨 뒤 원소의 인덱스
- 큐가 비어 있으면 일반적으로 `front = rear = -1`로 둔다.
- 삽입할 때는 `rear`가 1 증가하고, 삭제할 때는 `front`가 1 증가한다.

## 주요 메서드

- `offer()`: 큐에 데이터를 삽입한다. (`add()`도 가능)
- `poll()`: 맨 앞 데이터를 확인한 뒤 제거한다.
- `peek()`: 맨 앞 데이터를 확인만 한다.
- `isEmpty()`: 큐가 비어 있는지 확인한다.
- `size()`: 큐의 크기를 반환한다.

## Deque

`Deque`는 Double Ended Queue의 줄임말로, 앞과 뒤 양쪽에서 모두 삽입과 삭제가 가능하다.

## 버퍼

버퍼는 데이터를 한 곳에서 다른 곳으로 전송하는 동안, 그 데이터를 잠시 보관하는 메모리 영역이다.

- 버퍼는 FIFO 방식의 큐로 구현되는 경우가 많다.
