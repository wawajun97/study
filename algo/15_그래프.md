그래프 : 정점들의 집합과 이들을 연결하는 간선들의 집합으로 구성된 자료 구조
- 정점(vertex) : 그래프의 구성요소로 하나의 연결점
- 간선(dege) : 두 정점을 연결하는 선 -> 두 정점이 간선으로 연결되어 있으면 인접해 있다라고 표현
- 차수(degree) : 정점에 연결된 간선의 수

그래프 종류  
1. 무향그래프 : 방향이 없는 그래프 -> 양방향
2. 유향그래프 : 방향이 있는 그래프 -> 단방향
3. 가중치그래프 : 간선에 수치가 있는 그래프
4. 사이클이 없는 유향 그래프

완전 그래프 : 정점들에 대해 가능한 모든 간선들을 가진 그래프
- 모든 정점들이 최대로 간선을 가지는 그래프

부분 그래프 : 원래 그래프에서 일부의 정점이나 간선을 제외한 그래프

간선 표현 방법  
인접행렬 : N * N 배열을 이용해서 간선 정보를 저장
- i,j 값에 간선정보를 표현
인접리스트 : 각 정점마다 다른 정점으로 나가는 간선의 정보를 저장
- 2차원배열을 활용
- 공간의 효율성이 좋아서 주로 사용 -> 리스트를 사용하여 관계가 있는 애들만 저장해두기 때문
간선리스트 : 간선의 정보를 객체로 표현하여 리스트에 저장

간선의 유무를 행렬로 표현
1. N x N 배열 생성
2. 행 번호와 열 번호는 그래프의 정점에 대응
3. 두 정점이 인접해 있으면 1, 그렇지 않으면 0으로 표현
- 무향 그래프 : \ 대각선을 기준으로 대칭
- 유향 그래프 : 행의 합 = 진출 차수, 열의 합 = 진입 차수
