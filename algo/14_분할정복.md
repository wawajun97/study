# 분할정복

분할정복은 문제를 더 작은 문제로 나눈 뒤, 각각을 해결해서 원래 문제를 해결하는 방법이다.

- 일반적으로 재귀로 구현하지만, 반드시 재귀만 사용하는 것은 아니다.
- 반복문으로 구현할 수도 있다.

## 예시: 가짜 동전 찾기

- 여러 동전 중 하나의 가짜 동전이 진짜 동전보다 가볍다고 하자.
- 양팔저울을 사용해 가짜 동전을 찾을 수 있다.

1. 동전을 절반씩 나누어 무게를 잰다.
2. 더 가벼운 쪽만 다시 절반으로 나누어 무게를 잰다.
3. 가짜 동전을 찾을 때까지 반복한다.

## 예시: 거듭제곱

- `C^24 = C^12 * C^12`
- `C^12 = C^6 * C^6`
- ...
- `C^2 = C * C`

큰 문제를 절반 크기의 같은 문제로 나누어 계산할 수 있다.

```java
public static int func(int x, int n) {
    	if(n == 1) return x;
    	if(n % 2 == 0) {
    		int y = func(x,n/2); // x^n/2 * x^n/2 = x^n;
    		return y * y;
    	} else { // 홀수일 때
    		int y = func(x,(n-1)/2);
    		return y * y * x; // x^(n-1)/2 * x^(n-1)/2 * x = x^n;
    	}
    }
```

## 예시: 같은 색 공간 만들기

- `N x N` 공간을 4등분한다.
- 한 영역이 모두 같은 색이 아니면 다시 4등분한다.
- 모두 같은 색이 되거나 더 이상 나눌 수 없을 때까지 반복한다.
- 실제로 배열을 자르지 않고, 확인할 정사각형의 좌상단 좌표와 크기를 인자로 넘겨 재귀 호출한다.

```java
public static void cut(int r, int c, int size) {
		int sum = 0;
		for(int i = r;i<r+size;i++) {
			for(int j =c;j<c+size;j++) {
				sum += spaces[i][j]; //1이면 초록색 0이면 흰색
			}
		}

		if(sum == size * size) {
			++green;
		} else if(sum == 0) {
			++white;
		} else {
			int half = size/2;
			cut(r,c,half);
			cut(r,c + half,half);
			cut(r + half,c,half);
			cut(r + half,c + half,half);
		}

	}
```

## 이분 탐색(이진 탐색)

이분 탐색은 정렬된 데이터에서 구간을 절반씩 줄여 가며 답을 찾는 방법이다.

- 시간 복잡도는 `O(logN)`이다.
- Lower Bound, Upper Bound, 파라메트릭 서치로 확장할 수 있다.

### Lower Bound

- 목표값 이상이 처음 나타나는 위치를 찾는다.
- 즉, 타겟 이상인 값들 중 가장 작은 값의 인덱스를 의미한다.

### Upper Bound

- 목표값보다 큰 값이 처음 나타나는 위치를 찾는다.
- 즉, 타겟 초과인 값들 중 가장 작은 값의 인덱스를 의미한다.

### 파라메트릭 서치

- 최적화 문제를 `yes / no` 판정 문제로 바꿔서 이분 탐색으로 푸는 방법이다.
- 특정 배열 원소를 찾는 것이 아니라, 주어진 범위 안에서 조건을 만족하는 최적값을 찾을 때 사용한다.

### 기본 절차

1. 중앙 원소를 고른다.
2. 중앙 원소와 목표값을 비교한다.
3. 같으면 탐색을 종료한다.
4. 목표값이 더 작으면 왼쪽 절반, 더 크면 오른쪽 절반에서 다시 탐색한다.
5. 값을 찾을 때까지 반복한다.

## 예시: 병뚜껑 숫자 맞추기

- 술래만 숫자를 알고 있고, 나머지 사람이 그 숫자를 맞춘다.
- 숫자를 말할 때마다 술래는 `up` 또는 `down`으로 힌트를 준다.
- 탐색 범위를 절반씩 줄여 가는 이분 탐색 구조와 비슷하다.
