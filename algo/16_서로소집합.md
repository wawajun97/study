서로소 집합(상호배타 집합) : 교집합이 없는 두 집합  

서로소 집합 연산
1. Make-set(x) : x를 단일원소로 하는 단위서로소집합을 만듬
2. Find-Set(x) : x가 속한 집합 찾기 -> 대표자를 return
3. Union(x,y) : x가 속한 집합과 y가 속한 집합을 합침 (x가 대표자)

ex)
1. Make-set(x), Make-set(y), Make-set(a), Make-set(b)
2. Union(x,y), Union(a,b)
3. Find-set(y), Find-set(a)
4. Union(x,a)

서로소 집합을 표현하는 방법
1. 연결 리스트
- 헤드가 대표자
- 각 원소들은 대표원소를 가리키는 링크를 가짐(next와 rep 두개의 링크 필요)    
2. 트리
- 루트 노드가 대표자
- 자식 노드는 부모 노드를 가리킴
- 루트는 부모로 자기 자신을 가리킴  

서로소 집합 트리 구현 최적화
- union(x,a) 시 x에 a 트리를 붙이면 편향이진트리가 될 수 있다.

1. 트리의 높이를 계산하여 rank가 낮은 집합을 rank가 높은 집합에 붙이기 (union by rank)
2. Find-Set으로 불린 노드의 부모노드를 모두 root 노드로 변경 (path compression)
=> 1번과 2번을 모두 사용한다.
