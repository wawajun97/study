# 서로소 집합

서로소 집합(상호배타 집합)은 교집합이 없는 두 집합을 말한다.

## 기본 연산

1. `Make-set(x)`: 원소 `x` 하나만 포함하는 집합을 만든다.
2. `Find-set(x)`: `x`가 속한 집합의 대표자를 찾는다.
3. `Union(x, y)`: `x`가 속한 집합과 `y`가 속한 집합을 합친다.

## 예시

1. `Make-set(x)`, `Make-set(y)`, `Make-set(a)`, `Make-set(b)`
2. `Union(x, y)`, `Union(a, b)`
3. `Find-set(y)`, `Find-set(a)`
4. `Union(x, a)`

## 서로소 집합 표현 방법

### 1. 연결 리스트

- 헤드가 대표자가 된다.
- 각 원소는 대표 원소를 가리키는 링크를 가진다.
- 보통 `next`, `rep` 두 개의 링크가 필요하다.

### 2. 트리

- 루트 노드가 대표자이다.
- 자식 노드는 부모를 가리킨다.
- 루트는 부모로 자기 자신을 가리킨다.

## 트리 구현 최적화

`union(x, a)`를 반복할 때 한쪽으로만 계속 붙이면 편향된 트리가 될 수 있다.

1. `union by rank`: 높이가 낮은 트리를 높은 트리에 붙인다.
2. `path compression`: `Find-set` 호출 시 경로상의 부모를 모두 루트로 바꾼다.

이 두 가지를 함께 사용하면 매우 효율적으로 동작한다.

```java
public static int findSet(int a) {
		if(parents[a] == a) return a;

		return parents[a] = findSet(parents[a]);
	}

	public static void union(int a, int b) {
		int aRoot = findSet(a);
		int bRoot = findSet(b);

		if(aRoot == bRoot) return;

		parents[bRoot] = aRoot;
	}
```
